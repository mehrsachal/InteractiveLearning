# Interactive Learning Hub

**Mission:** To make learning fun and more accessible using visual interactions.

[Visit the Geospatial Visualisation Gallery](https://mehrsachal.github.io/InteractiveLearning/)

## About

This project is an interactive gallery built with React and Vite. It aims to explore complex photogrammetry and remote sensing concepts through interactive, physics-based simulations.

### Features
- Geospatial visualisations
- Search and filter by categories
- Dynamic GitHub profile integration

## Running Locally

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
